Projects

Here is the Javascript I used to create this website.

On the home page, there are two call action buttons that bubble out when your cursor goes over them. In order to achieve this effect, I used an event listener. Mouseover was used to scale the size of the button and mouseout was used to return the button back to the original size. I used querySelectorAll so that all buttons have this effect.

The navigation turns into a hamburger menu for mobile. I used an event listener to toggle the hamburger visibility when you are in mobile versus desktop. I also added a mouseleave event so that the navigation collapses when your mouse moves.

In the navigation, when you click on a link, it turns green. I used mouseover and mouseout event listener to achieve this effect. I used querySelectorAll so all the links (anchors) have this effect.

On the home page, there is a countdown clock that shows the days, hours, minutes and seconds until the next championship event. I created a variable with the date of the event and used getTime to get the date in seconds. I used the setInterval method to update the timer every second. I subtracted the countdown date from today’s date. Math.floor was used to calculate the days, hours, minutes and seconds.

On each page, the H1 has an animated text effect. I selected the element by ID, split the words using textContent.split and used a forEach to loop through each word. The opacity was set to 0 then goes to 1 so that the words go from invisible to visible. I used the setTimeout function to achieve this.

On the roster page, I used an array with name and weight class to create the list. I used a function to show the athlete list by pulling the list through the element’s (paragraph) ID. I used a forEach loop to loop through the list and create a string that appends to the innerHTML. I also used a window.onload function so that when the page fully loads, it displays the list of athletes.

On the contact form, I added form validation so that an alert would pop up if not all the fields were filled in. I used an event listener and a function that is triggered when the submit button is clicked. I also used the preventDefault method to stop the form from being submitted if all the required fields were empty.
